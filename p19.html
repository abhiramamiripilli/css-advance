CSS Pseudo-elements..!

he CSS Pseudo-elements allow you to style the elements or parts of the elements without adding any IDs or classes to them.

CSS3 introduced a new double-colon (::) syntax for Pseudo-elements to distinguish between the and pseudo-classes.

selector::Pseudo-element { property: values; }
