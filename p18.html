The CSS PSEUDO-CLASSES allow you to style the dynamic states of an element such as hover , active and focus state , as wella as elements that are existing in the documnet tree but can't be targated via the use of other selector with out adding any IDs or classes to them, for example,targeting the first or last child elements.

A PSEUDO-CLASSES starts with a colon (:). its syntax can be given with:

selector:pseudo-class {property: value; }
